
## Getting Started

Setup project environment with [virtualenv](https://virtualenv.pypa.io) and [pip](https://pip.pypa.io).

```bash
# Setup
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
# Run
$ python manage.py migrate
$ python manage.py runserver
```
Setup database 
```bash
# Setup
$ docker compose up -d
```
